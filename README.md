# 🚀 Super Hack Menu

<img src="logo.png" width="120" alt="Super Hack Menu Logo" />

**Super Hack Menu** is a powerful and customizable mod menu for Geometry Dash, built to enhance gameplay with advanced tools, cheats, and a modern interface.

> Designed for players, creators, and testers who want more control over their Geometry Dash experience.

---

## ✨ Features

- 🎮 Gameplay hacks (practice tools, speed control, modifiers)
- ⚙️ Customizable settings and keybinds
- 🧰 Utilities for level testing and debugging
- 🎨 Clean and modern UI
- 🌐 Vietnamese support (GDVN)
- 🚀 Lightweight and optimized

---

## 📥 Installation

1. Install **Geode Loader**
2. Download the mod
3. Put it into your `Geode/mods` folder
4. Launch Geometry Dash

---

## 📦 Requirements

- Geode Loader v5.6.1 or higher  
- Geometry Dash v2.2081  

### Required Mods
- GDVN (>= v5.0.1)  
- Custom Keybinds (>= v2.1.0)  
- Misc Bugfixes (>= v1.10.1)  

> Dependencies are automatically installed by Geode if missing.
