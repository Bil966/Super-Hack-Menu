
# 🚀 Super Hack Menu

**Super Hack Menu** is a powerful and customizable mod menu for Geometry Dash, designed to enhance gameplay with advanced tools, utilities, and a clean interface.

> Built for players, creators, and testers who want full control over their Geometry Dash experience.

---

## ✨ Features

- 🎮 Gameplay hacks (practice tools, speed control, modifiers)
- ⚙️ Fully customizable settings and keybinds
- 🧰 Utilities for level testing and debugging
- 🎨 Clean and modern UI design
- 🌐 Vietnamese support (GDVN)
- 🚀 Optimized for performance

---

## 📥 Installation

1. Install **Geode Loader**
2. Download **Super Hack Menu**
3. Place the mod file into your Geode mods folder
4. Launch Geometry Dash and enjoy!

---

## 📦 Requirements

- Geode Loader v5.6.1 or higher  
- Geometry Dash v2.2081  

### Required Mods
- GDVN (>= v5.0.1)  
- Custom Keybinds (>= v2.1.0)  
- Misc Bugfixes (>= v1.10.1)  

> These dependencies are automatically installed by Geode if missing.
